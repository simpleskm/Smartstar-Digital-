# Smartstar Digital Agency Website

A modern, responsive, and visually appealing website built for **Smartstar Digital Agency**, showcasing digital services, tutoring programs, chess coaching, portfolio projects, testimonials, and contact information.

## Overview

This project was designed to provide a professional online presence for Smartstar Digital Agency while delivering a seamless user experience across desktop, tablet, and mobile devices.

The website features a premium user interface, responsive navigation, interactive animations, and a clean layout that highlights the agency's services and achievements.

## Features

* Responsive design for all screen sizes
* Modern hero section with call-to-action buttons
* Interactive navigation menu
* Service showcase section
* Admissions section for tutoring and chess coaching
* Portfolio gallery
* Client testimonials
* Contact form with validation
* Smooth scrolling navigation
* Scroll-reveal animations
* Premium light-blue gradient theme
* Glassmorphism card design

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
## Getting Started

1. Clone or download the project.
2. Open the project folder.
3. Ensure all files are in their correct directories.
4. Open `index.html` in your browser.

No additional dependencies or installations are required.
